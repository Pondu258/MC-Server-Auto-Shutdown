# MC-Server-Auto-Shutdown

MCサーバーの停止後に自動的にPCのシャットダウンを行うプログラムです。  

## 目次

1. [使用想定](#使用想定)
2. [導入・使用方法](#方法)
3. [設定](#設定)
4. [更新情報](#更新情報)

## 使用想定

多人数でのマルチプレイ時に、鯖主が外出・就寝する際等の使用を想定しています。\
stopコマンドの権限譲渡、もしくはサーバーの停止を自動化するpluginやmod等と併用することをおすすめします。

## 導入・使用方法<a id="方法"></a>

### 導入

以下の図を参照して適当な位置に.exeを配置してください。

<pre>
📁 (どこでも可)
　├── MSAS-vx.x.x.exe（本プログラム）
　├── 📁 MSAS/（初回起動時自動生成）
　│　　├── config.json
　│　　└── launcher.log
　└── 📁 server/（サーバーフォルダ）
　　　　└── forge-server.jar（サーバーファイル）
</pre>

### 使用

サーバー起動の際、.jarの代わりに先ほどの.exeをダブルクリックで起動します。\
起動後はコンソールの指示に従ってください。設定終了後、自動的にサーバーが起動します。



## 設定
プログラム起動後のコンソールから設定が可能です。\
設定ファイルは、初期設定から変更時のみ.exeと同階層にconfig.jsonが生成されます。

### server_folder
サーバーフォルダを指定

### server_jar
サーバーファイル（.jar）を指定

### countdown_seconds
サーバー停止後にシャットダウンするまでの猶予時間（秒）

### shutdown_time_start&nbsp;・&nbsp;shutdown_time_end
サーバー停止後にシャットダウンを行う時間帯（24時間表記）
