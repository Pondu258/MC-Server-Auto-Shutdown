# MC-Server-Auto-Shutdown

MCサーバーの停止後に自動的にPCのシャットダウンを行うプログラムです。  

## 目次

1. [使用想定](#使用想定)
2. [導入・使用方法](#導入・使用方法)
3. [設定](#設定)

## 使用想定

多人数でのマルチプレイ時に、鯖主が外出・就寝する際等の使用を想定しています。\
stopコマンドの権限譲渡、もしくはサーバーの停止を自動化するmod等と併用することをおすすめします。

## 導入・使用方法

### 導入

ダウンロードした.zipを解凍。\
サーバーフォルダ内に.exeを配置。

📁 server（サーバーフォルダ）\
　├── MSAS-vx.x.x.exe（本プログラム）\
　├── server.jar（サーバーファイル）　\
　└── etc...

### 使用

サーバー起動の際、先ほどの.exeをダブルクリックで起動します。\
起動後はコンソールの指示に従ってください。設定終了後、自動的にサーバーが起動します。



## 設定
プログラム起動後のコンソールから設定が可能です。\
設定ファイルは、初期設定から変更時のみ.exeと同階層にconfig.jsonが生成されます。

### server_jar
サーバーファイル（.jar）を指定

### countdown_seconds
サーバー停止後にシャットダウンするまでの猶予時間（秒）
